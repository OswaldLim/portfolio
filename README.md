# Oswald Lim — Portfolio

A clean, minimal personal portfolio website. No build tools, no frameworks, just HTML, CSS, and a tiny bit of vanilla JS.

## Features
- ⚡ Zero dependencies, instant load
- 🌙 Dark mode toggle (respects OS preference)
- 📱 Fully responsive
- ♿ Accessible (semantic HTML, proper contrast)
- 🔧 Easy to update — edit `index.html` directly

## Structure
```
portfolio/
├── index.html          # Main page
├── assets/
│   ├── css/style.css   # All styles
│   └── js/main.js      # Theme toggle + smooth scroll
└── README.md           # This file
```

## How to Update Content
1. Open `index.html` in any text editor
2. Edit the placeholder text with your actual details
3. Replace the `#` links in project cards with real URLs
4. Save and deploy

## Deployment
- **Vercel**: `vercel deploy` (auto-deploys from Git)
- **Netlify**: Drag and drop the `portfolio/` folder
- **GitHub Pages**: Push to a repo and enable Pages
